🎮 HOW TO USE IN THUNDER CLIENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Open Thunder Client
  → Go to VS Code left sidebar
  → Click Thunder Client icon

Step 2: Create Request
  → Click "+" button to create new request
  → OR Import collection (recommended)

Step 3: Fill in Details
  → Method: Select GET or POST
  → URL: Copy from above
  → Headers: Add if needed (for POST)
  → Body: Copy JSON from above

Step 4: Send Request
  → Click blue "Send" button
  → Wait for response on right side

Step 5: View Response
  → Response appears in JSON format
  → Check Status: 200 = Success, 500 = Error

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🐛 IF YOU GET ERROR:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 500 Error on Generate:
   ✅ FIXED! - Database tables are now created
   ✅ Try again, should work now

❌ Server not responding:
   → Make sure Laravel server is running
   → Run: php artisan serve --host=0.0.0.0 --port=8000

❌ Invalid JSON in Body:
   → Check quotes are correct
   → Duration must be number, not string
   → No trailing commas

❌ Missing header "Content-Type":
   → For POST requests, add header
   → Key: Content-Type
   → Value: application/json
